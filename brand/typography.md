# BulenCoin Typography

- **Display / Headings**: `Space Grotesk` (weights 500–700)  
  - Use for H1–H3 and CTAs. Upper/lowercase as written; avoid all caps except short pills.
- **Body / UI**: `Manrope` (weights 400–600)  
  - Use for body copy, labels, nav, buttons.
- **Monospace**: `IBM Plex Mono` / system monospace fallback  
  - Use for code, addresses, seed/backup blocks.

Sizing (web reference):

- H1: 42–56px; H2: 32–40px; H3: 24–28px; body: 16–18px; small: 13–14px.

Rules:

- Keep line-height 1.2–1.35 for headings, 1.5–1.6 for body.
- Avoid mixing other type families; keep max two weights per block.
- Never stretch or skew the logo wordmark; maintain consistent letter spacing.
